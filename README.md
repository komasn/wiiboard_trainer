# wiiboard_trainer  

WiiバランスボードをBluetoothでLinux（Raspberry Piなど）に接続し、日々の宅トレ（自宅トレーニング）と体重管理をターミナル上で一元管理できるヘルスケア相棒アプリです。

ゲーム感覚で毎日の健康管理とトレーニング習慣の継続をサポートします。

## ✨ 主な機能

- **🔥 ランダム筋トレ指令**: 起動するたびに、スクワットやプランクなどのメニューとコツをランダムで提案します。
- **⚖️ 自動体重測定**: トレーニング完了後、Wiiバランスボードに乗るだけで自動的に体重を精密測定します。
- **💾 アクティビティ自動ロギング**: 日時、行ったトレーニング、測定された体重が自動的にCSVファイル（`health_log.csv`）へ1行で綺麗に蓄積されます。
- **💻 ターミナル完結**: 軽量・シンプルに動作し、GUI環境がなくても24時間サーバー化したラズパイなどから手軽に実行可能です。

## 👥 動作環境

- Linux環境 (Raspberry Pi 4等で動作確認済み)
- Python 3.x
- Wiiバランスボード (RVL-021)
- Bluetooth環境

## 🔧 セットアップ手順

### 1. 必要なパッケージのインストール
Linux環境にBluetooth関連のライブラリとPython用バインディングをインストールします。

```bash
sudo apt update
sudo apt install bluetooth bluez libbluetooth-dev pi-bluetooth -y
pip3 install pybluez

```
### 2. WiiバランスボードのBluetoothアドレス（BD_ADDR）の確認
Wiiバランスボードをペアリング待機状態（裏面の電池カバー内にある赤色の同期ボタンを押して、表面のシンクロLEDが点滅している状態）にします。
その状態でターミナルから以下のコマンドを実行し、ボードのアドレスを探します。
```bash
hcitool scan

```
実行結果に Nintendo RVL-WBC-01 と表示されたマックアドレス（例: 00:26:59:XX:XX:XX）をコピーしておきます。

### 3. ソースコードの修正
trainer/wiiboard_trainer.py を開き、31行目付近にある BD_ADDR を、先ほど確認したご自身のWiiバランスボードのマックアドレスに書き換えます。
```python
# --- ⚖️ Wiiバランスボード設定 ---
BD_ADDR = "あなたのWiiボードのマックアドレス"

```
## 🏃‍♂️ 使い方
 1. ターミナルからスクリプトを実行します。
   ```bash
   python3 trainer/wiiboard_trainer.py
   
   ```
 2. 画面に「今日の筋トレ指令」が表示されます。メニューとコツを確認して、トレーニングを行います。
 3. 運動が終了したら、キーボードの Enter キーを押します。
 4. 画面の指示に従い、Wiiバランスボードの裏面にある**赤色の同期ボタン**を1回ポンと押して接続します。
 5. 「準備完了！」と表示されたら、ボードの上に真っ直ぐ直立して乗ります。
 6. 数秒間、体重が安定すると自動的に測定が完了し、health_log.csv にデータが保存されます。
## 📊 記録の確認
蓄積されたトレーニングと体重の履歴は、生成されたCSVファイルからいつでも確認できます。
```bash
cat health_log.csv

```
**出力例:**
```text
日時,メニュー,体重(kg)
2026-06-23 18:17:56,プランク,55.5
2026-06-24 08:30:12,スクワット,55.6

```
## 💡 技術的な特徴（パケット安定化の工夫）
WiiバランスボードはBluetooth接続の初期段階において、パケットデータ（重心値）が大きく暴れる特性を持っています。本スクリプトでは、正確な「0点調整（キャリブレーション）」を行うため、**接続直後の最初の1000パケットを完全にスキップして通信を安定させた後に基準値を計算する**独自の安定化ロジックを実装しています。これによって非常に精密な体重測定を実現しています。
